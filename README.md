# FashionStore E-Commerce Website

A full-stack Java web application built using JSP, Servlets, JDBC, MySQL, HTML, CSS, and JavaScript.

## Features
- User Registration & Login
- Product Catalog
- Product Details
- Shopping Cart
- Checkout
- Order Management
- Session Handling
- Responsive UI

## Technologies
- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML/CSS
- JavaScript
- Apache Tomcat
