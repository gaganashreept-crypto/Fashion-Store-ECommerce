# FashionStore E-Commerce Website

FashionStore – Full Stack Java E-Commerce Website
Developed a responsive e-commerce web application using Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, and JavaScript following the MVC architecture. Implemented user authentication, product catalog, product details, shopping cart management, checkout process, order placement, and order history modules. Designed a modern UI and integrated database operations using DAO and DAO Implementation patterns.

## Features
- User Registration & Login
- Product Catalog
- Product Details
- Shopping Cart
- Checkout
- Order Management
- Session Handling
- Responsive UI

## Technologies
- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML/CSS
- JavaScript
- Apache Tomcat
